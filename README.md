# Netflix Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends in content, genres, release years, and country distribution.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Significant growth in Netflix content after 2015
- Most content belongs to Drama and International genres
- USA and India are among top content-producing countries

## 📁 Dataset
Dataset source: Kaggle Netflix dataset

## 📁 Project Structure
- netflix_analysis.ipynb → Main analysis notebook

## ▶️ How to Run
1. Open notebook in Jupyter
2. Run all cells
